## Hidzor-mac
状态栏图标隐藏工具，支持自定义图标。


![123](./image/README/show.gif)


## 项目截图
点击图标，圆点左侧的图标就会被隐藏。

## 功能
+ 点击状态栏图标隐藏/显示其他状态栏图标
+ 支持自定义 GIF 图标
+ 右键菜单切换不同图标

开发语言：python
